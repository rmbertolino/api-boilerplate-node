# api-boilerplate-node

### Requisitos previos
- Asegúrate de tener Docker y Docker Compose instalados en tu sistema.

### Pasos para configurar la base de datos
1. Clona este repositorio en tu máquina local.

2. Navega al directorio raíz del repositorio:

   ```bash
   cd MiApp

3. Crea un archivo .env en el directorio raíz para configurar las variables de entorno

4. Inicia el contenedor de la base de datos utilizando Docker Compose
    ```bash
    docker-compose up
